Introduction
++++++++++++
This a basic wiki application that allows you to create both public and private 
page.


Basic features
--------------

 - Shows all the public pages.
 - Displays private and public pages to loggedin user.
 - Ability to edit pages.
 - Creole markup language support.
 - Ability to show whole revision history of an article.
 - Compare any two version of an article.
 - Crosslink support.
